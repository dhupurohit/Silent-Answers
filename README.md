# Reflective Oracle

A minimal web app that offers reflective responses when a user silently holds a question in mind.

## Features

- Single-page interface
- Direct answer generator
- Krishna-inspired guidance generator
- Unlimited usage
- No backend required

## Tech Stack

HTML  
CSS  
Vanilla JavaScript

## How to Run

1. Download or clone the repo
2. Open index.html in a browser

## Deployment

Can be hosted on:

- GitHub Pages
- Netlify
- Vercel
